# 빌드

## 빌드 명령어

### Build

- 프로젝트 디렉토리에서 다음 명령어를 실행해 주십시오:

```bash
make -C applications/helloworld
```

![빌드 예시](images/build-output-1.png)
<br>
![빌드 예시](images/build-output-2.png)

### 빌드 출력

- 빌드가 성공하면 다음 파일들이 생성됩니다:

```
applications/helloworld/bin/SHIELD_M2354/
├── helloworld.elf          # ELF 파일
├── helloworld.bin          # 바이너리 파일 
├── helloworld.lh.bin       # 다운로드용 바이너리 파일
├── helloworld.hex          # HEX 파일
└── helloworld.map          # 메모리 맵 파일
```

### Clean

- 빌드된 파일들을 삭제하려면 다음 명령어를 실행해 주십시오:

```bash
make -C applications/helloworld clean
```

## 빌드 구조

프로젝트는 계층적 Makefile 구조를 사용합니다:

```
applications/helloworld/Makefile    # 애플리케이션 Makefile
Makefile.base                       # 기본 빌드 규칙
Makefile.include                    # 공통 설정
...
```
- 각 모듈별로 참조하면서 빌드가 진행됩니다.
