# Examples

## PQC

- Examples for running benchmarks and KAT(Known Answer Test) for PQC algorithms.

### How to run

```bash
> pqc
```

PQC example usage

![pqc main](../images/example-pqc-main.png)

```bash
> pqc bench list
```

List the algorithms available for PQC benchmark tests

![pqc bench list](../images/example-pqc-bench.png)

```bash
> pqc bench all
```

Run all PQC benchmark tests

![pqc bench all](../images/example-pqc-benchall.png)

```bash
> pqc kat list
```

List the algorithms available for PQC KAT tests

![pqc kat list](../images/example-pqc-kat.png)

```bash
> pqc kat all
```

Run all PQC KAT tests

![pqc kat all](../images/example-pqc-katall.png)
