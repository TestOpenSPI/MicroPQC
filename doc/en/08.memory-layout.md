# Memory Layout

Memory configuration of the M2354 chip:

| Category | Flash | RAM |
|---|---:|---:|
| **M2354 Total** | 1024KB | 256KB |
| **Secure firmware** | 320KB | 128KB |
| **User firmware (available)** | 704KB | 128KB |

## Detailed memory map

### Flash
- **0x0000_0000 ~ 0x0004_FFFF (320KB)**: Secure region (Bootloader + Secure firmware)
- **0x0005_0000 ~ 0x000F_FFFF (704KB)**: Non-Secure region (Application)

### RAM
- **0x2000_0000 ~ 0x2001_FFFF (128KB)**: Secure RAM
- **0x2002_8000 ~ 0x2003_FFFF (128KB)**: Non-Secure RAM
