# 모듈 설명

이 프로젝트는 모듈 단위로 빌드되며, 사용하고자 하는 모듈을 추가/삭제 또는 신규 생성하여 개발이 가능합니다.

## Application Makefile 설정

각 모듈은 모두 Application의 Makefile에서 설정합니다. 아래는 Makefile에 대한 설명입니다.
```makefile
# applications/helloworld/Makefile

BOARD ?= SHIELD_M2354 # 사용할 보드를 의미합니다. boards/SHIELD_M2354가 빌드에 포함됩니다.

OS = freertos # 사용할 RTOS를 설정합니다. FreeRTOS를 사용합니다.

TRUSTZONE = nonsecure # TrustZone은 Non-Secure 영역입니다. 고정입니다.

SECURE_FW = GENERIC_M2354_KPQC # 사용할 시큐어 펌웨어를 명시합니다.
SECURE_VER = V2.0.0 # 시큐어 펌웨어의 버전을 명시합니다. secure/v2.0.0/[BOARD]가 빌드에 포함됩니다.
SECURE_BANK = SINGLE # BANK 모드(DUAL or SINGLE)를 설정합니다. 현재 고정입니다.

# 사용할 모듈을 설정합니다.
USEMODULE += shell hexdump 
USEMODULE += axiocrypto_nonsecure_sdk
USEMODULE += axiocrypto_pqc_example
USEMODULE += spiflash mtd
USEMODULE += utils
USEMODULE += printf

INCLUDE_TESTLIB = 1 # 테스트 코드 포함 여부를 설정합니다. 각 모듈 폴더의 _test/가 빌드에 포함됩니다.

QUIET = 1 # 빌드 과정 출력 여부를 설정합니다. 0: 전체 출력, 1: 간단 출력
```

## Axiocrypto NonSecure SDK

암호 기능을 수행하는 모듈입니다.

- **위치**: `sys/axiocrypto_nonsecure_sdk/`
- axiocrypto.h를 참고해 주십시오.
- 별도의 AxioCrypto 관련 문서도 참고해 주십시오.

## Axiocrypto PQC

PQC 알고리즘 동작 테스트를 수행하는 모듈입니다.

- **위치**: `sys/axiocrypto_pqc_example/`

## Shell

명령줄 인터페이스를 제공하는 모듈입니다.

- **위치**: `sys/shell/`

## NSC Broker

Non-Secure 영역에서 Secure 영역의 함수를 호출하기 위한 브로커입니다.

- **위치**: `sys/nscbroker/`

## Hexdump

메모리 내용을 16진수로 덤프하는 유틸리티입니다.

- **위치**: `sys/hexdump/`

## ISRPipe

인터럽트 핸들러와 애플리케이션 간 데이터 전송을 위한 파이프입니다.

- **위치**: `sys/isrpipe/`

## Syscall

Newlib의 시스템 콜을 구현합니다.

- **위치**: `sys/syscall/`
- 변경하지 마십시오.

## Version

펌웨어 버전 정보를 제공합니다.

- **위치**: `sys/version/`
