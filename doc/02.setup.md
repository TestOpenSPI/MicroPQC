# 환경설정

## Toolchain 설정

### 1. ARM GCC Toolchain 다운로드

- https://developer.arm.com/downloads/-/gnu-rm 에서 **"14.3.rel1-x86_64-arm-none-eabi"** 버전을 다운로드해 주십시오.

- **파일명**: `arm-gnu-toolchain-14.3.rel1-x86_64-arm-none-eabi`
- **플랫폼**: Linux x86_64

### 2. Toolchain 경로 설정

- 프로젝트 내 `scripts/ARM_EABI_TOOLCHAIN.mk` 파일에서 toolchain 경로를 설정합니다.

```makefile
# Toolchain 경로를 실제 설치된 경로로 수정
ARM_EABI_TOOLCHAIN = /opt/arm-gnu-toolchain-14.3.rel1-x86_64-arm-none-eabi
```

![Toolchain 경로](images/toolchain-setup.png)

### 3. Toolchain 확인

- 설치가 올바르게 되었는지 확인해 주십시오:

```bash
/opt/arm-gnu-toolchain-14.3.rel1-x86_64-arm-none-eabi/bin/arm-none-eabi-gcc --version
```
