# 예제

## PQC

- PQC 알고리즘에 대한 Benchmark 및 KAT(Known Answer Test) 예제입니다.

### 실행 방법

```bash
> pqc
```

PQC 예제 사용법

![pqc main](images/example-pqc-main.png)

```bash
> pqc bench list
```

PQC Benchmark 테스트 알고리즘 리스트

![pqc bench list](images/example-pqc-bench.png)

```bash
> pqc bench all
```

PQC Benchmark 전체 테스트

![pqc bench all](images/example-pqc-benchall.png)

```bash
> pqc kat list
```

PQC KAT 테스트 알고리즘 리스트

![pqc kat list](images/example-pqc-kat.png)

```bash
> pqc kat all
```

PQC KAT 전체 테스트

![pqc kat all](images/example-pqc-katall.png)


### 소스 코드

```
sys/axiocrypto_pqc_example/
├── axiocrypto_pqc_example.c        # Example 메인
├── axiocrypto_pqc_bench.c          # Benchmark
├── axiocrypto_pqc_kat.c            # KAT
├── axiocrypto_pqc_common.c         # 공용 함수
├── axiocrypto_pqc_aimer128f.c      # AIMer128F 테스트
├── axiocrypto_pqc_dilithium2.c     # Dilithium2 테스트
├── axiocrypto_pqc_falcon512.c      # Falcon512 테스트
├── axiocrypto_pqc_haetae2.c        # HAETAE2 테스트
├── axiocrypto_pqc_kyber512.c       # Kyber512 테스트
├── axiocrypto_pqc_ntru768.c        # NTRU+768 테스트
├── axiocrypto_pqc_smaug1.c         # SMAUG-T1 테스트
└── axiocrypto_pqc_sphincs128f.c    # SPHINCS+128F 테스트
```
