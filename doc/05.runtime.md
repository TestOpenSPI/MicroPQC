# 실행화면

부팅이 완료되면 콘솔 UART를 통해 Shell 인터페이스가 실행됩니다.

## 부팅 시퀀스

1. **Bootloader**: Bootloader가 펌웨어 검증을 수행합니다.
2. **Secure**: 디바이스 인증서 및 펌웨어를 검증합니다.
3. **Non-Secure**: Board, FreeRTOS, Shell 등을 초기화합니다.
4. **Shell 시작**: UART Shell 인터페이스가 활성화됩니다.

## Shell 화면

- 부팅이 완료되면 콘솔 UART를 통해 다음과 같은 배너와 함께 Shell 프롬프트가 표시됩니다.

![실행화면](images/runtime-booting.png)

## 콘솔 UART 터미널 설정

- **Baud Rate**: 115200
- **Data Bits**: 8
- **Parity**: None
- **Stop Bits**: 1
- **Flow Control**: None

## Shell 명령어

- Shell 명령어를 통해 기능을 확인할 수 있습니다.
```
> help
Command              Description
---------------------------------------

reset                reset the board
fheap                print FreeRTOS heap status
fps                  print FreeRTOS process status
finfo                print FreeRTOS version
pqc                  AxioCrypto PQC Example
```
