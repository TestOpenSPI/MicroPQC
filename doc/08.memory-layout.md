# 메모리 레이아웃

M2354 칩의 메모리 구성:

| 구분 | Flash | RAM |
|------|-------|-----|
| **M2354 Total** | 1024KB | 256KB |
| **시큐어 펌웨어** | 320KB | 128KB |
| **유저 펌웨어 (사용 가능)** | 704KB | 128KB |

## 메모리 맵 상세

### Flash 메모리
- **0x0000_0000 ~ 0x0004_FFFF (320KB)**: Secure 영역 (Bootloader + Secure Firmware)
- **0x0005_0000 ~ 0x000F_FFFF (704KB)**: Non-Secure 영역 (Application)

### RAM 메모리
- **0x2000_0000 ~ 0x2001_FFFF (128KB)**: Secure RAM
- **0x2002_8000 ~ 0x2003_FFFF (128KB)**: Non-Secure RAM
